# <a name="wsl-2"></a>WSL 2

WSL 2 是 WSL 中体系结构的新版本, 它更改 Linux 发行版与 Windows 交互的方式。 WSL 2 的主要目标是提高文件系统性能并增加系统调用的完全兼容性。 每个 Linux 发行版都可以作为 WSL 1 或 WSL 2 发行版运行, 并可随时进行切换。 WSL 2 是底层体系结构的主要功能, 它使用虚拟化技术和 Linux 内核来实现其新功能。
